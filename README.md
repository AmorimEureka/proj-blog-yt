# Projeto Blogue YT

## Setup

1. Iniciar repositório git:

    ```
    git init
    ```

2. Criar ambiente virtual:

    ```
    python -m venv .venv
    ```


3. Criar arquivo .gitignore para projetos Django

4. Criar arquivo requirements.txt

    ```
    pip freeze >> requirements.txt
    ```
5. Instalar bibliotecas necessárias
